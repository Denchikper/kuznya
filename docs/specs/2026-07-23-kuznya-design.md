# Кузня — дизайн v2.0 (переписывание на Tauri)

## Суть

PyQt5-генератор паролей переписывается на Tauri v2 (macOS + Windows) под именем
«Кузня»: пароль «выковывается» из символов. Один экран, компактное окно.

## Архитектура

| Слой | Решение |
|------|---------|
| Оболочка | Tauri v2 |
| Бэкенд | Rust: команда `generate_password(opts)` с криптостойким RNG (OsRng), без modulo bias |
| Фронтенд | React 18 + TypeScript + Vite, нативный CSS с токенами |
| Буфер обмена | tauri-plugin-clipboard-manager |
| Вне Tauri | fallback на `crypto.getRandomValues` — дизайн можно смотреть в браузере |

## Функциональность

- Слайдер длины 4–64 (по умолчанию 16)
- Тумблеры: строчные, заглавные, цифры, символы; «исключить похожие» (l/1/I/O/0)
- Гарантия: в пароле есть хотя бы один символ каждого включённого набора
- Индикатор надёжности = «нагрев металла» + биты энтропии
- Копирование в один клик, подтверждение «Скопировано»

## Визуальный язык

- Тёмная тема: графит `#16171B`, стальные панели; светлая — холодная светлая
  сталь с синеватым отливом. Автоматически по системной теме
- Акцент — раскалённый металл: `#8C2B1A → #FF7A29 → #FFC15E → #FFF3D6`
- Шрифты вшиты (офлайн): Unbounded (заголовок), JetBrains Mono (пароль)
- Подпись: анимация «проковки» — символы перебираются и застывают слева направо;
  шкала нагрева: Холодный металл → Прогрев → Ковка → Белое каление
- `prefers-reduced-motion` отключает анимацию
